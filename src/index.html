Мои навыки

HTML5

Уверенное применение HTML5. Навыки семантически правильной и валидной (W3C validator) верстки, опыт кроссбраузерной (IE11, MS Edge, gecko, webkit, blink) и адаптивной (responsive) верстки, ретинизация. Также стараюсь верстать доступно.

CSS3

Уверенное применение CCS3. Верстка на flexbox, использование трансформаций, базовое владение анимациями. Также знаком с CSS Grid.

Sass

Уверенное владение и применение Sass. Пользуюсь упрощенным синтаксисом, но также знаком с SCSS. Из препроцессоров CSS пробовал работать и с less, но предпочтение отдал именно Sass.

JavaScript

Средние знания JavaScript. На данный момент активно углубляюсь именно в это направление, в частности в ES6. Делаю учебные проекты, смотрю вебинары, читаю книги и статьи по JS. На практике пишу велосипеды и стараюсь разбираться как те или иные вещи работают. Реализацию части полученных знаний можно увидеть на этом сайте.

Gulp

Опыт работы с Gulp.js. С помощью Gulp поднимаю локальный сервер, разделяю директории для разработки и готового результата, оптимизирую изображения, создаю SVG спрайты, компилирую sass в css, минифицирую и конкатенирую js файлы и в целом сильно упрощаю себе жизнь :)

Git

Базовые навыки работы с Git. Понимаю принципы и общую схему работы данной системы контроля версий.

Adobe Photoshop

Владение Adobe Photoshop на уровне работы с PSD макетами. Также иcпользую данную программу в процессе подготовки изображений для адаптивных сайтов. Для работы с SVG использую Adobe Illustrator.
